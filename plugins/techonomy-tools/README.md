# Techonomy Tools Plugin

Techonomy-branded skills for creating client deliverables.

## Skills

### campagneplan-presentatie

Generates a complete, Techonomy-branded campaign plan presentation (.pptx) based on a client briefing.

Triggers automatically when you ask Claude to create a campagneplan, paid advertising strategy deck, or media plan presentation.

**Example usage:**
```
"Maak een campagneplan presentatie voor KNVB, budget €25.000, kanalen Meta en Google Ads, periode april–mei 2026"
"Genereer een campagneplan deck voor client X"
"Ik wil een presentatie voor een paid advertising campagne"
```

The skill produces a ready-to-present .pptx file following Techonomy's exact branding, structure, and 4-chapter narrative arc (Inleiding → Campagne aanpak → Budget verdeling → Mediaplan).
